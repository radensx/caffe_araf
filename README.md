# Caffe Araf

Aplikasi kasir sederhana berbasis Java Swing.

## Fitur
- Login
- Kelola Pegawai
- Transaksi Penjualan
- Riwayat Transaksi
- Detail Transaksi

## Cara Instalasi
1. Clone repositori:
   `git clone https://github.com/radensx/caffe_araf`
2. Buka di NetBeans.
3. Import database `db_araf_caffe.sql` ke phpMyAdmin.
4. Jalankan `LoginForm.java`
